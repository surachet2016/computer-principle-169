บทที่ 9

การจัดการไฟล์

เรียนรู้การอ่านและเขียนไฟล์ด้วย Python ตั้งแต่การเปิดไฟล์ การอ่านข้อมูล การเขียนข้อมูล ไปจนถึงการจัดการข้อผิดพลาดที่อาจเกิดขึ้น — ทักษะพื้นฐานสำหรับโปรแกรมในชีวิตจริง

CLO3: สามารถเขียนโปรแกรมอ่านและเขียนข้อมูลจากไฟล์ได้อย่างถูกต้องและปลอดภัย

📁 1. การจัดการไฟล์คืออะไร?

การจัดการไฟล์ (File Handling) คือความสามารถของโปรแกรมในการอ่านและเขียนข้อมูลจากไฟล์บนดิสก์ ทำให้ข้อมูลคงอยู่ได้แม้ปิดโปรแกรมแล้ว

ในโปรแกรมจริงเราจำเป็นต้องใช้ไฟล์เสมอ เช่น:

  • บันทึกคะแนนนักศึกษาลงไฟล์ CSV
  • อ่านการตั้งค่าจากไฟล์ config.txt
  • เก็บประวัติการทำรายการลงใน log file
  • นำเข้าข้อมูลจาก dataset ขนาดใหญ่
Text File vs Binary File
ประเภทเนื้อหาตัวอย่างmode ใน Python
Text Fileข้อความที่อ่านได้.txt, .csv, .py, .html'r', 'w', 'a'
Binary Fileข้อมูลดิบ (bytes).jpg, .pdf, .mp3, .zip'rb', 'wb'

บทนี้จะเน้น Text File ซึ่งใช้บ่อยที่สุดในการเขียนโปรแกรมระดับเริ่มต้น

🔓 2. การเปิดและปิดไฟล์

ใช้ฟังก์ชัน open(filename, mode) เพื่อเปิดไฟล์ ซึ่งจะคืน file object กลับมา และต้อง close() ทุกครั้งหลังใช้งานเสร็จเพื่อปล่อยทรัพยากร

ตาราง Mode สำหรับ open()
Modeความหมายถ้าไฟล์ไม่มีถ้าไฟล์มีอยู่แล้ว
'r'Read (อ่านอย่างเดียว)เกิด Errorอ่านจากต้นไฟล์
'w'Write (เขียนอย่างเดียว)สร้างไฟล์ใหม่ลบแล้วเขียนใหม่ทั้งหมด
'a'Append (ต่อท้าย)สร้างไฟล์ใหม่เขียนต่อท้ายไฟล์เดิม
'r+'Read + Writeเกิด Errorอ่านและเขียนได้
'w+'Write + Readสร้างไฟล์ใหม่ลบแล้วเขียนใหม่
Python 
# วิธีเปิดและปิดไฟล์แบบดั้งเดิม (ต้อง close() เอง)
f = open("example.txt", "w")
f.write("Hello, File!")
f.close()   # สำคัญมาก — ถ้าลืมอาจทำให้ข้อมูลหาย

# ตรวจสอบว่าปิดแล้วหรือยัง
print(f.closed)   # True

# เปิดพร้อมระบุ encoding สำหรับภาษาไทย
f = open("thai.txt", "w", encoding="utf-8")
f.write("สวัสดีครับ")
f.close()
ข้อควรระวัง: อย่าลืม close()

ถ้าไม่ close() ไฟล์อาจเกิดปัญหาได้ เช่น ข้อมูลไม่ถูกบันทึกลงดิสก์ (ยังค้างใน buffer) หรือไฟล์ถูก lock ไม่ให้โปรแกรมอื่นใช้งาน ในทางปฏิบัติแนะนำให้ใช้ with statement แทน (ดูหัวข้อที่ 5)

📖 3. การอ่านไฟล์

เมื่อเปิดไฟล์ด้วย mode 'r' แล้ว สามารถอ่านข้อมูลได้หลายวิธี:

  • read() — อ่านทั้งไฟล์เป็น String เดียว
  • readline() — อ่านทีละบรรทัด คืน String (รวม \n)
  • readlines() — อ่านทุกบรรทัดคืนเป็น List of Strings
  • for line in file — วนลูปอ่านทีละบรรทัด (ประหยัดหน่วยความจำ)
Python 
# สมมติมีไฟล์ students.txt มีเนื้อหา:
# สมชาย,85
# สมหญิง,92
# สมศักดิ์,78

# วิธีที่ 1: read() — อ่านทั้งหมดในครั้งเดียว
with open("students.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# วิธีที่ 2: readline() — อ่านทีละบรรทัด
with open("students.txt", "r", encoding="utf-8") as f:
    line1 = f.readline()   # "สมชาย,85\n"
    line2 = f.readline()   # "สมหญิง,92\n"
    print(line1.strip())   # ลบ \n ออก

# วิธีที่ 3: readlines() — ได้ List ของบรรทัด
with open("students.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    print(lines)           # ['สมชาย,85\n', 'สมหญิง,92\n', ...]

# วิธีที่ 4: for loop — แนะนำสำหรับไฟล์ใหญ่
with open("students.txt", "r", encoding="utf-8") as f:
    for i, line in enumerate(f, 1):
        name, score = line.strip().split(",")
        print(f"{i}. {name}: {score} คะแนน")

✏️ 4. การเขียนไฟล์

เปิดไฟล์ด้วย mode 'w' (เขียนใหม่ทั้งหมด) หรือ 'a' (ต่อท้าย) แล้วใช้:

  • write(string) — เขียน string ลงไฟล์ (ไม่มี newline อัตโนมัติ ต้องใส่ \n เอง)
  • writelines(list) — เขียนหลายบรรทัดจาก List (ยังต้องใส่ \n เองด้วย)
Python 
# write() — เขียนทีละบรรทัด
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("บรรทัดที่ 1\n")
    f.write("บรรทัดที่ 2\n")
    f.write("บรรทัดที่ 3\n")

# writelines() — เขียนจาก List (ต้องใส่ \n เอง!)
lines = ["สมชาย,85\n", "สมหญิง,92\n", "สมศักดิ์,78\n"]
with open("students.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)

# mode 'a' — ต่อท้ายโดยไม่ลบเนื้อหาเดิม
with open("students.txt", "a", encoding="utf-8") as f:
    f.write("วิมล,96\n")    # เพิ่มนักศึกษาคนใหม่

# เขียนข้อมูลที่คำนวณแล้ว
students = [("สมชาย", 85), ("สมหญิง", 92), ("สมศักดิ์", 78)]
with open("report.txt", "w", encoding="utf-8") as f:
    f.write("=== รายงานคะแนน ===\n")
    for name, score in students:
        f.write(f"{name}: {score} คะแนน\n")
    avg = sum(s for _, s in students) / len(students)
    f.write(f"เฉลี่ย: {avg:.2f}\n")

🛡️ 5. with Statement (Context Manager)

with statement คือวิธีที่แนะนำที่สุดสำหรับการจัดการไฟล์ใน Python เพราะจะ close() ไฟล์ให้อัตโนมัติเมื่อออกจากบล็อก with ไม่ว่าจะเกิด exception หรือไม่ก็ตาม

รูปแบบ: with open(filename, mode) as f:

เปรียบเทียบ: แบบดั้งเดิม vs with statement
แบบดั้งเดิมwith statement
ปิดไฟล์ต้อง close() เองอัตโนมัติ
Exceptionclose() อาจถูกข้ามหาก errorclose() เสมอ
โค้ดยาวกว่ากระชับ อ่านง่าย
แนะนำไม่แนะนำแนะนำ
Python 
# รูปแบบที่แนะนำ: with statement
with open("notes.txt", "w", encoding="utf-8") as f:
    f.write("สร้างไฟล์ด้วย with statement\n")
    f.write("ปิดไฟล์อัตโนมัติ\n")
# ออกจาก with block → ไฟล์ถูกปิดทันที

print(f.closed)   # True

# เปิด 2 ไฟล์พร้อมกัน (อ่านและเขียน)
with open("input.txt", "r", encoding="utf-8") as fin, \
     open("output.txt", "w", encoding="utf-8") as fout:
    for line in fin:
        fout.write(line.upper())  # แปลงเป็นตัวพิมพ์ใหญ่

# ตัวอย่างจริง: บันทึกคะแนนลงไฟล์
students = [
    ("สมชาย", 85),
    ("สมหญิง", 92),
    ("สมศักดิ์", 78),
    ("วิมล", 96),
]

with open("scores.txt", "w", encoding="utf-8") as f:
    f.write("ชื่อ,คะแนน,เกรด\n")
    for name, score in students:
        grade = "A" if score >= 90 else "B" if score >= 80 else "C"
        f.write(f"{name},{score},{grade}\n")

print("บันทึกข้อมูลสำเร็จ!")

⚠️ 6. การจัดการข้อผิดพลาด

การเปิดไฟล์อาจเกิดข้อผิดพลาดได้หลายสาเหตุ เช่น ไฟล์ไม่มีอยู่ ไม่มีสิทธิ์เข้าถึง หรือดิสก์เต็ม ควรใช้ try/except รับมือกับข้อผิดพลาดเหล่านี้เสมอ

Exception ที่พบบ่อยเมื่อจัดการไฟล์
Exceptionสาเหตุ
FileNotFoundErrorไม่พบไฟล์ที่ระบุ (mode 'r')
PermissionErrorไม่มีสิทธิ์อ่านหรือเขียนไฟล์
IsADirectoryErrorระบุ path เป็นโฟลเดอร์แทน
OSErrorข้อผิดพลาดระดับระบบปฏิบัติการ (ครอบคลุมทั้งหมด)
UnicodeDecodeErrorEncoding ไม่ตรงกับไฟล์จริง
Python 
# จัดการ FileNotFoundError
def read_file_safe(filename):
    try:
        with open(filename, "r", encoding="utf-8") as f:
            content = f.read()
        return content
    except FileNotFoundError:
        print(f"ไม่พบไฟล์: {filename}")
        return None
    except PermissionError:
        print(f"ไม่มีสิทธิ์อ่านไฟล์: {filename}")
        return None
    except UnicodeDecodeError:
        print(f"Encoding ไม่ถูกต้อง ลองเปลี่ยน encoding")
        return None

result = read_file_safe("missing.txt")
if result is None:
    print("ไม่สามารถอ่านไฟล์ได้")

# ตรวจสอบว่าไฟล์มีอยู่ก่อนเปิด
import os

filename = "data.txt"
if os.path.exists(filename):
    with open(filename, "r", encoding="utf-8") as f:
        print(f.read())
else:
    print(f"ไฟล์ {filename} ยังไม่มีอยู่ กำลังสร้างใหม่...")
    with open(filename, "w", encoding="utf-8") as f:
        f.write("ข้อมูลเริ่มต้น\n")
