การวนซ้ำ (Loops)

เรียนรู้การวนซ้ำด้วย for loop และ while loop การควบคุม loop ด้วย break และ continue การใช้ else กับ loop การวนซ้ำซ้อน (Nested Loop) พร้อมตัวอย่างโปรแกรมจริง

🔹 การวนซ้ำคืออะไร และทำไมต้องใช้?

ทำไมต้องใช้การวนซ้ำ (Loop)?

ลองนึกภาพว่าถ้าต้องการพิมพ์ตัวเลข 1 ถึง 100 โดยไม่ใช้ loop เราต้องเขียน print(1), print(2), ... ไปถึง print(100) นั่นคือ 100 บรรทัด! แต่ด้วย loop เราสามารถเขียนได้ใน 2-3 บรรทัดเท่านั้น

การวนซ้ำ (Loop) ใช้ทำงานซ้ำๆ โดยไม่ต้องเขียนโค้ดซ้ำ ประหยัดเวลา ลดข้อผิดพลาด และทำให้โค้ดสั้นและอ่านง่าย Python มี loop 2 ชนิดหลัก คือ for loop และ while loop

Python 
# ไม่ใช้ loop (ซ้ำซาก และผิดพลาดง่าย)
print(1)
print(2)
print(3)
# ... ต้องเขียนถึง print(100) !!

# ใช้ loop (กระชับ)
for i in range(1, 101):
    print(i)
🔹 for loop

for loop ใน Python

for loop ใช้วนซ้ำตามจำนวนครั้งที่กำหนด หรือวนผ่านสมาชิกใน sequence (เช่น list, string, range) โดยอัตโนมัติ

for ตัวแปร in sequence:
    คำสั่งที่จะทำในแต่ละรอบ

ผังงาน for loop:

Python 
# for loop พื้นฐาน
for i in range(1, 6):
    print(f"รอบที่ {i}")

# range() มีรูปแบบต่างๆ
print("\n--- range(5) ---")
for i in range(5):          # 0, 1, 2, 3, 4
    print(i, end=" ")

print("\n\n--- range(1, 10, 2) ---")
for i in range(1, 10, 2):  # 1, 3, 5, 7, 9 (step=2)
    print(i, end=" ")

print("\n\n--- range(10, 0, -1) ---")
for i in range(10, 0, -1): # นับถอยหลัง
    print(i, end=" ")
print()

# วนผ่านสมาชิกใน list
fruits = ["แอปเปิล", "กล้วย", "ส้ม", "มะม่วง"]
print("\nรายการผลไม้:")
for fruit in fruits:
    print(f"  - {fruit}")

enumerate() — วนพร้อมดึง index

ฟังก์ชัน enumerate() ช่วยให้ดึงทั้ง index และค่าออกมาพร้อมกันในการวนซ้ำ

Python 
# enumerate() ดึง index + ค่าพร้อมกัน
subjects = ["คณิตศาสตร์", "ภาษาอังกฤษ", "วิทยาศาสตร์", "ประวัติศาสตร์"]

for index, subject in enumerate(subjects, start=1):
    print(f"{index}. {subject}")

# แทนที่จะเขียนแบบเก่า:
# for i in range(len(subjects)):
#     print(f"{i+1}. {subjects[i]}")
🔹 while loop

while loop

while loop วนซ้ำตราบเท่าที่เงื่อนไขยังเป็น True ต่างจาก for loop ที่รู้จำนวนรอบล่วงหน้า while loop เหมาะเมื่อไม่รู้ว่าจะต้องวนกี่รอบ ต้องระวังไม่ให้เกิด Infinite Loop (วนไม่หยุด)

while เงื่อนไข:
    คำสั่งที่จะทำซ้ำ
    (ต้องมีบางอย่างที่ทำให้เงื่อนไขเป็น False ในที่สุด)

ผังงาน while loop:

Python 
# while loop พื้นฐาน
count = 0
while count < 5:
    print(f"นับ: {count}")
    count += 1   # สำคัญ! ต้องอัปเดต count มิฉะนั้นจะ loop ไม่หยุด
print("จบการนับ")

# while loop: ถามซ้ำจนกว่าจะกรอกถูก (จำลอง)
attempts = 0
max_attempts = 3
correct_pin = "1234"
guesses = ["0000", "9999", "1234"]  # จำลอง input

while attempts < max_attempts:
    guess = guesses[attempts]
    attempts += 1
    print(f"พยายามครั้งที่ {attempts}: กรอก {guess}")
    if guess == correct_pin:
        print("รหัสถูกต้อง! เข้าสู่ระบบสำเร็จ")
        break
    else:
        remaining = max_attempts - attempts
        if remaining > 0:
            print(f"รหัสผิด เหลืออีก {remaining} ครั้ง")
        else:
            print("ล็อกบัญชีชั่วคราว")
⚠️ ระวัง Infinite Loop!
ถ้าลืมอัปเดตตัวแปรในเงื่อนไข while loop จะวนไม่หยุด เช่น while count < 5: แต่ไม่มี count += 1 จะวนไปตลอด หากเกิดเหตุการณ์นี้ให้กด Ctrl+C เพื่อหยุดโปรแกรม
🔹 break — หยุด loop

คำสั่ง break

คำสั่ง break ใช้หยุด loop ทันทีและออกจาก loop เมื่อพบเงื่อนไขที่ต้องการ แม้ว่าเงื่อนไขของ loop ยังเป็น True อยู่ก็ตาม

Python 
# break: ค้นหาสินค้า
products = ["ดินสอ", "ปากกา", "สมุด", "ไม้บรรทัด", "กระดาษ"]
target = "สมุด"
found = False

for product in products:
    print(f"ตรวจสอบ: {product}")
    if product == target:
        found = True
        break   # พบแล้ว หยุดค้นหา

if found:
    print(f"พบ '{target}' ในรายการ!")
else:
    print(f"ไม่พบ '{target}'")

# break ใน while loop
print("\nหาตัวเลขหารด้วย 7 ลงตัวตัวแรกใน range 1-50:")
for n in range(1, 51):
    if n % 7 == 0:
        print(f"พบ: {n}")
        break
🔹 continue — ข้ามรอบ

คำสั่ง continue

คำสั่ง continue ใช้ข้ามไปยังรอบถัดไปทันที โดยไม่ทำคำสั่งที่เหลือในรอบนั้น (ต่างจาก break ที่หยุด loop ทั้งหมด)

Python 
# continue: แสดงเฉพาะเลขคี่ใน 1-10
print("เลขคี่ 1-10:")
for n in range(1, 11):
    if n % 2 == 0:
        continue    # ข้ามเลขคู่ ไปรอบถัดไป
    print(n, end=" ")
print()

# continue: แสดงเฉพาะเลขที่หาร 3 ไม่ลงตัว
print("\nเลข 1-20 ที่ไม่ใช่ผลคูณของ 3:")
for n in range(1, 21):
    if n % 3 == 0:
        continue    # ข้ามผลคูณของ 3
    print(n, end=" ")
print()

# ใช้ break และ continue ร่วมกัน
print("\nเลขใน 1-20 ที่เป็นเลขคี่และหาร 5 ไม่ลงตัว (หยุดถ้า > 15):")
for n in range(1, 21):
    if n > 15:
        break
    if n % 2 == 0 or n % 5 == 0:
        continue
    print(n, end=" ")
print()
🔹 else กับ loop

else ใน for/while loop

Python มีฟีเจอร์พิเศษที่ภาษาอื่นไม่ค่อยมี คือ else ใน loop บล็อก else จะทำงานเมื่อ loop สิ้นสุดตามปกติ (ไม่ถูกหยุดด้วย break) มีประโยชน์มากในการค้นหาข้อมูล

Python 
# else กับ for loop: ค้นหาจำนวนเฉพาะ
def is_prime_check(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            # พบตัวหาร แสดงว่าไม่ใช่จำนวนเฉพาะ
            break
    else:
        # loop จบโดยไม่ถูก break = หาตัวหารไม่ได้ = จำนวนเฉพาะ
        return True
    return False

# ทดสอบ
test_nums = [2, 3, 4, 7, 9, 11, 15, 17]
for num in test_nums:
    result = "จำนวนเฉพาะ" if is_prime_check(num) else "ไม่ใช่จำนวนเฉพาะ"
    print(f"{num:3d}: {result}")
🔹 Nested Loop (การวนซ้ำซ้อน)

การวนซ้ำซ้อน

เราสามารถมี loop ซ้อนอยู่ภายใน loop ได้ เรียกว่า Nested Loop สำหรับทุกรอบของ loop นอก loop ในจะทำงานครบทุกรอบ จำนวนรอบทั้งหมดคือผลคูณของจำนวนรอบแต่ละชั้น

Python 
# ตารางสูตรคูณ 1-3
print("ตารางสูตรคูณ 1-3:")
print("-" * 35)
for i in range(1, 4):
    for j in range(1, 11):
        print(f"{i} × {j:2d} = {i*j:3d}", end="   ")
    print()  # ขึ้นบรรทัดใหม่หลังจบแต่ละแถว
print("-" * 35)

# วาดรูปสามเหลี่ยม
print("\nสามเหลี่ยมดาว:")
rows = 5
for i in range(1, rows + 1):
    for j in range(i):
        print("*", end=" ")
    print()

# วาดรูปสี่เหลี่ยม
print("\nสี่เหลี่ยมดาว 4×5:")
for i in range(4):
    for j in range(5):
        print("*", end=" ")
    print()

🔹 ตัวอย่างโปรแกรมจริง
Python 
# ตัวอย่างที่ 1: หาผลรวม 1-100
total = 0
for i in range(1, 101):
    total += i
print(f"ผลรวมของ 1 ถึง 100 = {total}")   # 5050

# ตัวอย่างที่ 2: หาจำนวนเฉพาะ 1-50
print("\nจำนวนเฉพาะที่น้อยกว่าหรือเท่ากับ 50:")
primes = []
for n in range(2, 51):
    is_prime = True
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            is_prime = False
            break
    if is_prime:
        primes.append(n)
print(primes)
print(f"จำนวนทั้งหมด: {len(primes)} จำนวน")

# ตัวอย่างที่ 3: ตารางสูตรคูณแม่ 7
print("\nสูตรคูณแม่ 7:")
for j in range(1, 13):
    print(f"7 × {j:2d} = {7*j:3d}")

💡 เมื่อไรใช้ for? เมื่อไรใช้ while?
  • for loop: ใช้เมื่อรู้จำนวนรอบล่วงหน้า หรือต้องการวนผ่านสมาชิกใน sequence เช่น list, string, range
  • while loop: ใช้เมื่อไม่รู้จำนวนรอบ แต่มีเงื่อนไขหยุด เช่น รอผู้ใช้กรอกค่าถูก รอการเชื่อมต่อ ฯลฯ
🔹 คำถามทบทวน

คำถามทบทวนบทที่ 5

  1. อธิบายความแตกต่างระหว่าง for loop และ while loop ใน Python พร้อมยกตัวอย่างสถานการณ์ที่เหมาะกับแต่ละชนิด และเขียนโค้ดตัวอย่างสำหรับแต่ละชนิด
  2. อธิบายความแตกต่างระหว่าง break และ continue จงเขียนโค้ดที่แสดงผลลัพธ์ที่ต่างกันเมื่อใช้ break กับเมื่อใช้ continue ในลูปเดียวกัน
  3. จงเขียนโปรแกรม Python ที่หาผลคูณของเลข 1 ถึง 10 (10 factorial = 10!) โดยใช้ for loop และแสดงค่าทีละขั้นตอน
  4. อธิบายว่า else ของ loop ทำงานอย่างไร และแตกต่างจาก else ของ if อย่างไร พร้อมยกตัวอย่างที่มีประโยชน์
  5. จงเขียนโปรแกรม Nested Loop ที่แสดงตาราง ดังนี้: แถว 1: * | แถว 2: * * | แถว 3: * * * | ... | แถว 5: * * * * * (รูปสามเหลี่ยม) และอธิบายว่า loop แต่ละชั้นทำหน้าที่อะไร
