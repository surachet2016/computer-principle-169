คำสั่งเงื่อนไข

เรียนรู้การควบคุมการทำงานของโปรแกรมด้วยคำสั่งเงื่อนไข if, if-else, if-elif-else รวมถึง Nested if, Ternary Operator และค่า Truthy/Falsy เพื่อให้โปรแกรมตัดสินใจได้อย่างชาญฉลาด

🔹 โครงสร้างควบคุม (Control Structures)

โครงสร้างควบคุมโปรแกรม 3 ประเภท

โปรแกรมคอมพิวเตอร์ทำงานโดยอาศัยโครงสร้างควบคุม 3 ประเภทหลัก ซึ่งรวมกันได้ทุกอัลกอริทึมในโลก:

  • Sequential (ตามลำดับ): ทำงานทีละคำสั่งตามลำดับจากบนลงล่าง เป็นค่าเริ่มต้นของโปรแกรม
  • Selection (การเลือก): ตัดสินใจว่าจะทำงานส่วนไหน โดยอาศัยเงื่อนไข (if, if-else, if-elif-else)
  • Repetition (การวนซ้ำ): ทำงานซ้ำๆ จนกว่าเงื่อนไขจะหยุด (for, while) — ศึกษาในบทที่ 5
🔹 คำสั่ง if

Syntax ของ if

คำสั่ง if ใช้ตรวจสอบเงื่อนไข ถ้าเงื่อนไขเป็น True จะทำงานในบล็อกนั้น ถ้าเป็น False จะข้ามไป

if เงื่อนไข:
    คำสั่งที่จะทำเมื่อเงื่อนไขเป็น True
    (อาจมีหลายบรรทัด)
⚠️ Indentation สำคัญมากใน Python!
Python ใช้การเยื้อง (Indentation) เพื่อกำหนดขอบเขตของบล็อกโค้ด ต่างจากภาษาอื่นที่ใช้วงเล็บปีกกา { } โดยมาตรฐานคือเยื้อง 4 ช่องว่าง (4 spaces) ต้องเยื้องให้เท่ากันทุกบรรทัดในบล็อกเดียวกัน มิฉะนั้นจะเกิด IndentationError
Python 
# คำสั่ง if พื้นฐาน
age = 18

if age >= 18:
    print("คุณมีสิทธิ์ลงคะแนนเสียงเลือกตั้ง")
    print("กรุณาไปใช้สิทธิ์ของคุณ")

# ถ้า age < 18 บรรทัดข้างในจะไม่ถูกรัน
print("โปรแกรมทำงานต่อไป")  # บรรทัดนี้รันเสมอ

# ตัวอย่าง: ตรวจสอบคะแนนผ่าน/ไม่ผ่าน
score = 72
if score >= 50:
    print(f"คะแนน {score}: ผ่าน")
🔹 คำสั่ง if-else

Syntax ของ if-else

if-else เพิ่มทางเลือกที่สอง: ถ้าเงื่อนไขเป็น False จะทำงานในบล็อก else

if เงื่อนไข:
    คำสั่งเมื่อเงื่อนไขเป็น True
else:
    คำสั่งเมื่อเงื่อนไขเป็น False
Python 
# if-else: ตรวจสอบเลขคู่/คี่
number = 7

if number % 2 == 0:
    print(f"{number} เป็นเลขคู่")
else:
    print(f"{number} เป็นเลขคี่")

# if-else: ตรวจสอบสิทธิ์
age = 15

if age >= 18:
    print("อนุญาตให้เข้าชม")
else:
    print(f"ขออภัย อายุ {age} ปี ยังไม่ถึงเกณฑ์")

# if-else: หาค่าสูงสุด
a, b = 42, 37
if a > b:
    maximum = a
else:
    maximum = b
print(f"ค่าสูงสุดระหว่าง {a} และ {b} คือ {maximum}")

🔹 คำสั่ง if-elif-else

Syntax ของ if-elif-else

เมื่อมีเงื่อนไขมากกว่า 2 ทางเลือก ใช้ elif (ย่อจาก else if) เพื่อเพิ่มเงื่อนไขได้ไม่จำกัด Python จะตรวจสอบตามลำดับจากบนลงล่าง และหยุดที่เงื่อนไขแรกที่เป็น True เท่านั้น

Python 
# ตรวจสอบเกรด A/B/C/D/F
score = 85

if score >= 80:
    grade = "A"
elif score >= 70:
    grade = "B"
elif score >= 60:
    grade = "C"
elif score >= 50:
    grade = "D"
else:
    grade = "F"

print(f"คะแนน {score} ได้เกรด {grade}")

# ตรวจสอบหมู่เลือด (ตัวอย่างหลายเงื่อนไข)
blood_type = "A"

if blood_type == "A":
    compatible = "A, O"
elif blood_type == "B":
    compatible = "B, O"
elif blood_type == "AB":
    compatible = "A, B, AB, O"
elif blood_type == "O":
    compatible = "O"
else:
    compatible = "ไม่ทราบ"

print(f"หมู่เลือด {blood_type} รับเลือดจากหมู่: {compatible}")
🔹 การซ้อน if (Nested if)

Nested if คืออะไร?

การซ้อน if หมายถึงการมีคำสั่ง if ซ้อนอยู่ภายในบล็อกของ if อีกชั้นหนึ่ง ใช้เมื่อต้องการตรวจสอบเงื่อนไขหลายระดับ แต่ควรระวังไม่ให้ซ้อนกันหลายชั้นเกินไป เพราะจะทำให้อ่านโค้ดยากขึ้น

Python 
# Nested if: ระบบสมัครงาน
age = 25
has_degree = True
years_experience = 3

if age >= 22:
    print("ผ่านเกณฑ์อายุ")
    if has_degree:
        print("ผ่านเกณฑ์วุฒิการศึกษา")
        if years_experience >= 2:
            print("ผ่านเกณฑ์ประสบการณ์")
            print("=> ยินดีรับเข้าทำงาน!")
        else:
            print(f"ประสบการณ์ไม่พอ ต้องการอย่างน้อย 2 ปี (มี {years_experience} ปี)")
    else:
        print("ต้องมีวุฒิการศึกษาระดับปริญญาตรี")
else:
    print(f"อายุน้อยเกินไป ต้องอายุ 22 ปีขึ้นไป (ปัจจุบัน {age} ปี)")
💡 เคล็ดลับ: หลีกเลี่ยง Nested if ลึกเกินไป
ถ้า nested if ลึกเกิน 3 ชั้น โค้ดจะอ่านยากมาก ลองใช้ and รวมเงื่อนไข หรือแยกเป็นฟังก์ชัน เช่น แทนที่จะซ้อน 3 ชั้น ใช้ if age >= 22 and has_degree and years_experience >= 2: แทน
🔹 Ternary Operator (Conditional Expression)

Ternary Operator

Python มีวิธีเขียน if-else แบบสั้นในบรรทัดเดียว เรียกว่า Ternary Operator หรือ Conditional Expression ใช้เมื่อต้องการกำหนดค่าให้ตัวแปรโดยขึ้นกับเงื่อนไข

ตัวแปร = ค่าถ้า_True  if  เงื่อนไข  else  ค่าถ้า_False
Python 
# Ternary Operator
x = 10
result = "บวก" if x > 0 else "ลบหรือศูนย์"
print(result)    # บวก

# เทียบกับ if-else ปกติ (ผลเหมือนกัน)
if x > 0:
    result2 = "บวก"
else:
    result2 = "ลบหรือศูนย์"
print(result2)   # บวก

# ตัวอย่างเพิ่มเติม
age = 20
status = "ผู้ใหญ่" if age >= 18 else "เยาวชน"
print(f"อายุ {age}: {status}")

score = 55
passed = "ผ่าน" if score >= 50 else "ไม่ผ่าน"
print(f"คะแนน {score}: {passed}")

# Ternary ใน f-string
n = 7
print(f"{n} เป็นเลข{'คู่' if n % 2 == 0 else 'คี่'}")
🔹 ค่า Truthy และ Falsy

Truthy / Falsy ใน Python

ใน Python ค่าทุกค่าสามารถใช้ในเงื่อนไข if ได้ โดยถูกแปลงเป็น bool โดยอัตโนมัติ

ค่า Falsy (ถือว่าเป็น False) ได้แก่:

  • False
  • 0 (ศูนย์ทุกรูปแบบ: 0, 0.0, 0j)
  • None
  • "" (string ว่าง)
  • [] (list ว่าง), () (tuple ว่าง), {} (dict/set ว่าง)

ค่า Truthy: ทุกค่าที่ไม่ใช่ค่า Falsy ข้างต้น

Python 
# Truthy / Falsy
values = [0, 1, -1, "", "hello", None, [], [1, 2], 0.0, 3.14]

for v in values:
    if v:
        print(f"{repr(v):15} → Truthy")
    else:
        print(f"{repr(v):15} → Falsy")

# ประยุกต์ใช้งาน
name = ""    # สมมุติ input จากผู้ใช้

if name:     # ตรวจสอบว่ามีค่าหรือไม่ (ง่ายกว่า name != "")
    print(f"สวัสดี คุณ{name}")
else:
    print("กรุณากรอกชื่อ")

🔹 ตัวอย่างโปรแกรมจริง: ตรวจสอบปีอธิกสุรทิน

เงื่อนไขปีอธิกสุรทิน (Leap Year)

ปีอธิกสุรทิน (Leap Year) คือปีที่มี 366 วัน โดยมีเงื่อนไข ดังนี้:

  • หารด้วย 4 ลงตัว และ ไม่หารด้วย 100 ลงตัว หรือ
  • หารด้วย 400 ลงตัว
Python 
# ตรวจสอบปีอธิกสุรทิน
year = 2024

if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"ปี {year} เป็นปีอธิกสุรทิน (366 วัน)")
else:
    print(f"ปี {year} ไม่ใช่ปีอธิกสุรทิน (365 วัน)")

# ทดสอบหลายปี
test_years = [2000, 1900, 2024, 2023, 2100, 2400]
print("\nตรวจสอบหลายปี:")
for y in test_years:
    is_leap = (y % 4 == 0 and y % 100 != 0) or (y % 400 == 0)
    result = "อธิกสุรทิน" if is_leap else "ปกติ"
    print(f"ปี {y}: {result}")

🔹 คำถามทบทวน

คำถามทบทวนบทที่ 4

  1. ทำไม Indentation (การเยื้อง) จึงสำคัญมากใน Python? จงอธิบายและยกตัวอย่างโค้ดที่ถูกต้องและผิดพลาดเรื่อง Indentation
  2. อธิบายความแตกต่างระหว่าง if-else และ if-elif-else พร้อมยกตัวอย่างสถานการณ์ที่เหมาะกับแต่ละรูปแบบ
  3. จงเขียนโปรแกรม Python ที่รับคะแนน (0-100) แล้วแสดงเกรดตามระบบ A, B+, B, C+, C, D+, D, F โดยกำหนดเกณฑ์เองอย่างสมเหตุสมผล
  4. จงอธิบายแนวคิด Truthy และ Falsy ใน Python พร้อมยกตัวอย่างค่า Falsy อย่างน้อย 5 ค่า และบอกว่าสามารถนำมาใช้ประโยชน์อย่างไรในการเขียนโค้ด
  5. จงเขียน Ternary Operator สำหรับโปรแกรมที่รับตัวเลขแล้วบอกว่าเป็นเลขคู่หรือเลขคี่ และอธิบายว่า Ternary Operator เหมาะกับสถานการณ์ใดบ้าง
