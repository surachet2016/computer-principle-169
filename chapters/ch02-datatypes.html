ชนิดข้อมูลและตัวแปร

เรียนรู้การใช้งานตัวแปร ชนิดข้อมูลพื้นฐานของ Python ได้แก่ int, float, str, bool และ None รวมถึงการแปลงชนิดข้อมูล เพื่อสร้างพื้นฐานที่แข็งแกร่งสำหรับการเขียนโปรแกรม

CLO1
🔹 ตัวแปร (Variable)

ตัวแปรคืออะไร?

ตัวแปร (Variable) คือพื้นที่ในหน่วยความจำ (Memory) ที่ใช้สำหรับเก็บข้อมูลชั่วคราวระหว่างที่โปรแกรมกำลังทำงาน เราสามารถตั้งชื่อให้กับตัวแปรเพื่อเรียกใช้ข้อมูลนั้นได้ในภายหลัง เปรียบเสมือนกล่องที่มีป้ายชื่อติดอยู่ ซึ่งเราสามารถนำของใส่ไว้ในกล่องและดึงออกมาใช้ได้ตามต้องการ

ใน Python การกำหนดค่าให้ตัวแปรทำด้วยเครื่องหมาย = (เครื่องหมายกำหนดค่า ไม่ใช่เครื่องหมายเท่ากับทางคณิตศาสตร์)

กฎการตั้งชื่อตัวแปร (Naming Rules)

  • ต้องขึ้นต้นด้วยตัวอักษร (a-z, A-Z) หรือเครื่องหมาย underscore (_)
  • ตัวถัดไปอาจเป็นตัวอักษร ตัวเลข หรือ underscore ก็ได้
  • ห้ามมีช่องว่าง (space) ในชื่อตัวแปร
  • ห้ามใช้คำสงวน (Keywords) ของ Python เช่น if, for, while, print
  • Python แยกตัวพิมพ์เล็ก-ใหญ่ (Case-sensitive): name กับ Name คือตัวแปรคนละตัว
  • แนะนำใช้รูปแบบ snake_case: เช่น student_name, total_score
Python 
# ตัวอย่างชื่อตัวแปรที่ถูกต้อง
name = "สมชาย"
age = 20
_score = 95.5
student_id = "6501001"

# ตัวอย่างชื่อตัวแปรที่ผิด (จะเกิด SyntaxError)
# 1name = "ผิด"       # ขึ้นต้นด้วยตัวเลขไม่ได้
# my name = "ผิด"    # มีช่องว่างไม่ได้
# if = 5             # ใช้ keyword ไม่ได้

# Python แยกตัวพิมพ์เล็ก-ใหญ่
myVar = 10
myvar = 20
MyVar = 30
print(myVar, myvar, MyVar)  # 10 20 30 (คนละตัวแปรกัน!)
🔹 ชนิดข้อมูลพื้นฐาน (Basic Data Types)

Python มีชนิดข้อมูลพื้นฐานที่สำคัญ 5 ชนิด ดังตาราง:

ชนิดข้อมูล คำอธิบาย ตัวอย่าง
int จำนวนเต็ม (Integer) ไม่มีทศนิยม 10, -5, 0, 2024
float จำนวนทศนิยม (Floating-point) 3.14, -2.5, 1.0
str ข้อความหรือสตริง (String) "สวัสดี", 'Hello'
bool ค่าความจริง (Boolean): True หรือ False True, False
None ค่าว่าง ไม่มีค่า (Null value) None
🔹 int (จำนวนเต็ม)

ชนิดข้อมูล int

ชนิดข้อมูล int (Integer) ใช้เก็บจำนวนเต็ม ทั้งบวกและลบ รวมถึงศูนย์ ใน Python 3 ตัวแปร int สามารถเก็บค่าได้ไม่จำกัดขนาด (ขึ้นอยู่กับหน่วยความจำของเครื่อง) ซึ่งต่างจากภาษาอื่นที่มักมีขีดจำกัด

Python 
# ตัวอย่าง int
age = 21
temperature = -5
year = 2024
big_number = 1_000_000  # ใช้ _ คั่นหลักพันได้ (อ่านง่ายขึ้น)

print(age)           # 21
print(temperature)   # -5
print(big_number)    # 1000000

# การคำนวณกับ int
a = 17
b = 5
print(a + b)    # 22
print(a - b)    # 12
print(a * b)    # 85
print(a // b)   # 3  (หารเอาจำนวนเต็ม)
print(a % b)    # 2  (เศษจากการหาร)
print(a ** 2)   # 289 (17 ยกกำลัง 2)
🔹 float (จำนวนทศนิยม)

ชนิดข้อมูล float

ชนิดข้อมูล float ใช้เก็บจำนวนที่มีทศนิยม หรือจำนวนที่เขียนในรูปแบบ Scientific Notation เช่น 1.5e3 (หมายถึง 1500.0) ควรระวังความแม่นยำของ float เพราะคอมพิวเตอร์เก็บค่า float ด้วยระบบเลขฐานสอง ซึ่งอาจเกิดความคลาดเคลื่อนเล็กน้อยได้

Python 
# ตัวอย่าง float
height = 1.75
pi = 3.14159
temperature = -12.5
avogadro = 6.022e23  # 6.022 × 10²³

print(height)       # 1.75
print(pi)           # 3.14159
print(avogadro)     # 6.022e+23

# ความแม่นยำของ float (ระวัง!)
result = 0.1 + 0.2
print(result)           # 0.30000000000000004 (ไม่ใช่ 0.3 พอดี!)
print(round(result, 2)) # 0.3 (ใช้ round() แก้ปัญหาได้)

# การหารปกติได้ผลเป็น float เสมอ
print(10 / 3)   # 3.3333333333333335
print(10 / 2)   # 5.0 (ไม่ใช่ 5)
⚠️ ระวัง
การหารด้วย / ใน Python 3 จะได้ผลเป็น float เสมอ แม้ว่าผลลัพธ์จะเป็นจำนวนเต็มก็ตาม เช่น 10 / 2 ได้ 5.0 ไม่ใช่ 5 หากต้องการผลเป็น int ให้ใช้ // แทน
🔹 str (สตริง)

ชนิดข้อมูล str (String)

ชนิดข้อมูล str ใช้เก็บข้อความหรือลำดับของตัวอักษร สามารถใช้เครื่องหมายคำพูดเดี่ยว (') หรือคู่ (") ครอบข้อความได้ทั้งคู่ สำหรับข้อความหลายบรรทัดใช้เครื่องหมายคำพูดสามชั้น (''' หรือ """)

Python 
# ตัวอย่าง str
name = "สุรเชษฐ์"
university = 'มหาวิทยาลัยนราธิวาสราชนครินทร์'
quote = "เขากล่าวว่า 'สวัสดี'"    # ใช้ " ครอบ ' ข้างในได้

# สตริงหลายบรรทัด
address = """123 ถนนหลัก
ตำบลกะลุวอ
อำเภอเมือง
จังหวัดนราธิวาส"""

print(name)
print(university)
print(address)

# การต่อสตริง (Concatenation)
first = "สวัสดี"
second = " โลก"
combined = first + second
print(combined)    # สวัสดี โลก

# การทำซ้ำสตริง
line = "-" * 20
print(line)        # --------------------

# ความยาวของสตริง
print(len(name))   # จำนวนตัวอักษร
🔹 bool (บูลีน)

ชนิดข้อมูล bool (Boolean)

ชนิดข้อมูล bool มีค่าได้เพียงสองค่าเท่านั้น คือ True (จริง) และ False (เท็จ) ต้องเขียนขึ้นต้นด้วยตัวพิมพ์ใหญ่เสมอ ใช้งานบ่อยในการตรวจสอบเงื่อนไข bool จริงๆ แล้วเป็น subclass ของ int โดย True = 1 และ False = 0

Python 
# ตัวอย่าง bool
is_student = True
has_passed = False
is_morning = True

print(is_student)    # True
print(has_passed)    # False

# bool จากการเปรียบเทียบ
age = 18
print(age >= 18)     # True
print(age > 20)      # False
print(age == 18)     # True
print(age != 18)     # False

# bool คือ int
print(True + True)   # 2
print(True + False)  # 1
print(int(True))     # 1
print(int(False))    # 0
🔹 None (ค่าว่าง)

ชนิดข้อมูล None

None ใช้แทนการไม่มีค่า หรือค่าที่ยังไม่ถูกกำหนด คล้ายกับ null ในภาษาอื่น มักใช้เมื่อต้องการประกาศตัวแปรแต่ยังไม่มีค่า หรือเมื่อฟังก์ชันไม่ส่งค่ากลับ

Python 
# ตัวอย่าง None
result = None
print(result)           # None
print(type(result))     # 

# ตรวจสอบว่าเป็น None หรือไม่ (ใช้ is ไม่ใช้ ==)
if result is None:
    print("ยังไม่มีผลลัพธ์")
else:
    print(f"ผลลัพธ์คือ: {result}")
🔹 การตรวจสอบชนิดข้อมูล — type()

Python มีฟังก์ชันในตัว type() ที่ใช้ตรวจสอบว่าตัวแปรนั้นเป็นชนิดข้อมูลอะไร มีประโยชน์มากในการดีบักโปรแกรม

Python 
# ตรวจสอบชนิดข้อมูลด้วย type()
name = "สุรเชษฐ์"
age = 25
height = 1.75
is_student = True
nothing = None

print(type(name))       # 
print(type(age))        # 
print(type(height))     # 
print(type(is_student)) # 
print(type(nothing))    # 

# isinstance() ตรวจสอบแบบยืดหยุ่นกว่า
print(isinstance(age, int))    # True
print(isinstance(age, float))  # False
print(isinstance(True, int))   # True (bool เป็น subclass ของ int)

🔹 การแปลงชนิดข้อมูล (Type Conversion)

การแปลงชนิดข้อมูล (Type Casting)

บางครั้งเราต้องการเปลี่ยนชนิดข้อมูลจากชนิดหนึ่งไปเป็นอีกชนิดหนึ่ง Python มีฟังก์ชันในตัวสำหรับทำสิ่งนี้ เรียกว่า Type Conversion หรือ Type Casting

  • int(x) — แปลง x ให้เป็นจำนวนเต็ม
  • float(x) — แปลง x ให้เป็นจำนวนทศนิยม
  • str(x) — แปลง x ให้เป็นสตริง
  • bool(x) — แปลง x ให้เป็น bool
Python 
# แปลงชนิดข้อมูล

# str → int
x = "42"
y = int(x)
print(y + 8)         # 50 (ถ้าไม่แปลง จะเกิด TypeError)

# float → int (ตัดทศนิยมทิ้ง ไม่ปัด)
pi = 3.99
print(int(pi))       # 3 (ไม่ปัดขึ้นเป็น 4)

# int → float
num = 5
print(float(num))    # 5.0

# int/float → str
score = 95.5
text = str(score)
print(text + " คะแนน")    # 95.5 คะแนน

# แปลงค่าต่างๆ เป็น bool
print(bool(0))       # False
print(bool(1))       # True
print(bool(-5))      # True (ทุกตัวเลขที่ไม่ใช่ 0 = True)
print(bool(""))      # False (string ว่าง = False)
print(bool("abc"))   # True

# input() คืนค่าเป็น str เสมอ จึงต้องแปลงก่อนใช้คำนวณ
# num = int(input("กรอกตัวเลข: "))  # แปลง str เป็น int
⚠️ ระวัง
ไม่สามารถแปลงสตริงที่ไม่ใช่ตัวเลขให้เป็น int หรือ float ได้ เช่น int("hello") จะเกิด ValueError ควรตรวจสอบข้อมูลให้แน่ใจก่อนแปลงเสมอ
🔹 ค่าคงที่ (Constant)

ค่าคงที่ใน Python

Python ไม่มีคีย์เวิร์ดสำหรับประกาศค่าคงที่โดยตรง แต่ตามแนวปฏิบัติมาตรฐาน (PEP 8) นิยมตั้งชื่อตัวแปรที่ใช้เป็นค่าคงที่ด้วยตัวพิมพ์ใหญ่ทั้งหมด (ALL_CAPS) โดยใช้ underscore คั่นคำ เพื่อบอกให้โปรแกรมเมอร์คนอื่นรู้ว่าไม่ควรเปลี่ยนค่านี้

Python 
# ค่าคงที่ (ตามแนวปฏิบัติ ALL_CAPS)
PI = 3.14159265358979
MAX_SCORE = 100
MIN_AGE = 18
COURSE_CODE = "07-034-205"
GRAVITY = 9.81  # m/s²

# ใช้ค่าคงที่ในการคำนวณ
radius = 5
area = PI * radius ** 2
print(f"พื้นที่วงกลมรัศมี {radius} = {area:.2f}")

# ข้อมูลนักศึกษา
student_name = "สมชาย ใจดี"
student_score = 88

if student_score <= MAX_SCORE:
    print(f"{student_name} ได้คะแนน {student_score}/{MAX_SCORE}")
