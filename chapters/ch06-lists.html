List / Array

เรียนรู้การจัดการข้อมูลหลายค่าในตัวแปรเดียวด้วย List ของ Python — โครงสร้างข้อมูลที่ใช้บ่อยที่สุดในการเขียนโปรแกรม

CLO3: เขียนโปรแกรมจัดการข้อมูลด้วย List ได้

📦 1. List คืออะไร?

List คือโครงสร้างข้อมูลที่ใช้เก็บข้อมูลหลายค่าในตัวแปรเดียว โดยเรียงลำดับกันและสามารถแก้ไขได้ (mutable)

ในภาษาอื่น เช่น C หรือ Java จะเรียกว่า Array แต่ใน Python จะเรียกว่า List ซึ่งมีความยืดหยุ่นมากกว่า เพราะสามารถเก็บข้อมูลต่างชนิดกันในตัวเดียวกันได้

  • เก็บข้อมูลหลายค่าเรียงลำดับ (ordered)
  • สามารถเปลี่ยนแปลงค่าได้ (mutable)
  • อนุญาตให้มีค่าซ้ำกัน (allow duplicates)
  • เข้าถึงข้อมูลด้วย index เริ่มจาก 0
  • รองรับข้อมูลหลายชนิดในตัวเดียว (mixed types)
💡 ทำไมต้องใช้ List?

แทนที่จะสร้างตัวแปร score1, score2, score3 ... จนถึง score100 เราสามารถเก็บทุกคะแนนไว้ใน List เดียวชื่อ scores แล้วใช้ index เข้าถึงแต่ละค่าได้ ทำให้โค้ดสั้น อ่านง่าย และจัดการได้สะดวกยิ่งขึ้น

🔨 2. การสร้าง List

สร้าง List ได้หลายวิธี:

  • ใช้วงเล็บเหลี่ยม [] — วิธีที่นิยมที่สุด
  • ใช้ฟังก์ชัน list() — แปลงจากข้อมูลประเภทอื่น
  • List เปล่า [] — สร้างก่อนแล้วค่อยเพิ่มข้อมูลภายหลัง
Python 
# List ของผลไม้ (String)
fruits = ["แอปเปิล", "กล้วย", "ส้ม", "มะม่วง"]

# List ของตัวเลข (Integer)
numbers = [1, 2, 3, 4, 5]

# List แบบผสมชนิดข้อมูล (Mixed Types)
mixed = [1, "hello", 3.14, True, None]

# List เปล่า
empty_list = []

# สร้างจาก list()
chars = list("Python")    # ['P', 'y', 't', 'h', 'o', 'n']
from_range = list(range(1, 6))  # [1, 2, 3, 4, 5]

print(fruits)
print(numbers)
print(mixed)
print(chars)
print(from_range)

🎯 3. การเข้าถึง Element (Indexing & Slicing)

ใช้ index ในวงเล็บเหลี่ยมเพื่อเข้าถึงสมาชิกใน List

  • Positive index: นับจากซ้ายเริ่มที่ 0 เช่น list[0], list[1]
  • Negative index: นับจากขวาเริ่มที่ -1 เช่น list[-1] คือตัวสุดท้าย
  • Slicing: list[start:end:step] ดึงข้อมูลหลายค่าพร้อมกัน
Index 0 1 2 3
ค่า "แอปเปิล" "กล้วย" "ส้ม" "มะม่วง"
Negative Index -4 -3 -2 -1
Python 
fruits = ["แอปเปิล", "กล้วย", "ส้ม", "มะม่วง", "สับปะรด"]

# Positive index
print(fruits[0])   # แอปเปิล (ตัวแรก)
print(fruits[2])   # ส้ม (ตัวที่ 3)

# Negative index
print(fruits[-1])  # สับปะรด (ตัวสุดท้าย)
print(fruits[-2])  # มะม่วง (ตัวรองสุดท้าย)

# Slicing [start:end] — ไม่รวม end
print(fruits[1:3])   # ['กล้วย', 'ส้ม']
print(fruits[:3])    # ['แอปเปิล', 'กล้วย', 'ส้ม'] (จากต้นถึง index 2)
print(fruits[2:])    # ['ส้ม', 'มะม่วง', 'สับปะรด'] (จาก index 2 ถึงท้าย)

# Slicing พร้อม step
print(fruits[::2])   # ['แอปเปิล', 'ส้ม', 'สับปะรด'] (ทุก 2 ตัว)
print(fruits[::-1])  # กลับลำดับทั้งหมด

✏️ 4. การแก้ไข List

List ใน Python เป็น mutable หมายความว่าสามารถแก้ไข เพิ่ม หรือลบสมาชิกได้หลังจากสร้างแล้ว

วิธีคำอธิบายตัวอย่าง
list[i] = valเปลี่ยนค่าที่ index ifruits[0] = "ชมพู่"
append(val)เพิ่มค่าที่ท้าย Listfruits.append("ลิ้นจี่")
insert(i, val)แทรกค่าที่ตำแหน่ง ifruits.insert(1, "มะละกอ")
remove(val)ลบค่าที่ระบุ (ค่าแรกที่พบ)fruits.remove("กล้วย")
pop(i)ลบและคืนค่าที่ index i (ไม่ระบุ = ตัวสุดท้าย)fruits.pop()
del list[i]ลบค่าที่ index idel fruits[0]
Python 
fruits = ["แอปเปิล", "กล้วย", "ส้ม"]
print("เริ่มต้น:", fruits)

# เปลี่ยนค่า
fruits[0] = "ชมพู่"
print("หลังเปลี่ยน [0]:", fruits)

# เพิ่มที่ท้าย
fruits.append("มะม่วง")
print("หลัง append:", fruits)

# แทรกที่ตำแหน่ง 1
fruits.insert(1, "มะละกอ")
print("หลัง insert:", fruits)

# ลบด้วยชื่อ
fruits.remove("ส้ม")
print("หลัง remove:", fruits)

# ลบตัวสุดท้าย
popped = fruits.pop()
print(f"pop ได้: {popped}, เหลือ: {fruits}")

# ลบด้วย del
del fruits[0]
print("หลัง del [0]:", fruits)

🛠️ 5. เมธอดของ List

Python มีเมธอด (method) สำเร็จรูปมากมายสำหรับจัดการ List

เมธอดคำอธิบาย
sort()เรียงลำดับ List (ascending โดยปริยาย)
sort(reverse=True)เรียงลำดับจากมากไปน้อย
reverse()กลับลำดับ List
index(val)หา index ของค่าที่ระบุ
count(val)นับจำนวนครั้งที่ค่าปรากฏ
copy()สำเนา List (shallow copy)
clear()ลบสมาชิกทั้งหมด
extend(list2)ต่อ List อื่นเข้ามา
Python 
scores = [85, 72, 90, 68, 95, 85, 78]

print("ต้นฉบับ:", scores)
print(f"index ของ 90: {scores.index(90)}")
print(f"นับจำนวน 85: {scores.count(85)}")

# เรียงลำดับ
scores.sort()
print("หลัง sort:", scores)

scores.sort(reverse=True)
print("sort จากมากไปน้อย:", scores)

# กลับลำดับ
scores.reverse()
print("หลัง reverse:", scores)

# สำเนา
copy_scores = scores.copy()
copy_scores.append(100)
print("ต้นฉบับ:", scores)    # ไม่เปลี่ยน
print("สำเนา:", copy_scores) # เปลี่ยนเฉพาะสำเนา

🔄 6. การวนซ้ำใน List

สามารถวนซ้ำผ่านสมาชิกใน List ได้หลายวิธี:

  • for item in list — วนผ่านค่าแต่ละตัว
  • for i, item in enumerate(list) — ได้ทั้ง index และค่า
  • for i in range(len(list)) — ใช้เมื่อต้องการ index
Python 
fruits = ["แอปเปิล", "กล้วย", "ส้ม", "มะม่วง"]

# วิธีที่ 1: for loop ธรรมดา
print("=== วิธีที่ 1 ===")
for fruit in fruits:
    print(fruit)

# วิธีที่ 2: enumerate (ได้ทั้ง index และค่า)
print("\n=== วิธีที่ 2: enumerate ===")
for i, fruit in enumerate(fruits):
    print(f"{i+1}. {fruit}")

# วิธีที่ 3: range + len
print("\n=== วิธีที่ 3: range+len ===")
for i in range(len(fruits)):
    print(f"fruits[{i}] = {fruits[i]}")

⚡ 7. List Comprehension

List Comprehension คือวิธีสร้าง List ใหม่จาก iterable ในบรรทัดเดียว ทำให้โค้ดกระชับและอ่านง่าย

รูปแบบ: [expression for item in iterable if condition]

Python 
# แบบปกติ (หลายบรรทัด)
squares_normal = []
for x in range(1, 6):
    squares_normal.append(x ** 2)
print("แบบปกติ:", squares_normal)

# แบบ List Comprehension (บรรทัดเดียว)
squares = [x**2 for x in range(1, 6)]
print("Comprehension:", squares)  # [1, 4, 9, 16, 25]

# พร้อมเงื่อนไข (if)
evens = [x for x in range(1, 21) if x % 2 == 0]
print("เลขคู่:", evens)

# แปลงชนิดข้อมูล
words = ["hello", "world", "python"]
upper_words = [w.upper() for w in words]
print("ตัวพิมพ์ใหญ่:", upper_words)

# กรองข้อมูล
scores = [85, 45, 92, 38, 76, 55, 98]
passing = [s for s in scores if s >= 60]
print("คะแนนผ่าน (>=60):", passing)
💡 เคล็ดลับ

ใช้ List Comprehension เมื่อโค้ดอ่านง่าย แต่หากมีเงื่อนไขซับซ้อนหลายชั้น ควรใช้ for loop ธรรมดาจะอ่านง่ายกว่า

🗂️ 8. List ซ้อน (Nested List)

Nested List คือ List ที่มี List อื่นเป็นสมาชิก ใช้แทน 2D array หรือ matrix ในคณิตศาสตร์

เข้าถึงด้วย double index: matrix[row][col]

Python 
# Matrix 3x3
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# เข้าถึงค่า
print(f"matrix[0][0] = {matrix[0][0]}")  # 1 (แถวที่ 0, คอลัมน์ที่ 0)
print(f"matrix[1][2] = {matrix[1][2]}")  # 6 (แถวที่ 1, คอลัมน์ที่ 2)
print(f"matrix[2][1] = {matrix[2][1]}")  # 8

# วนซ้ำ matrix
print("\n=== แสดง Matrix ===")
for row in matrix:
    for val in row:
        print(val, end="\t")
    print()

# คะแนนนักศึกษา 3 คน, 3 วิชา
student_scores = [
    [85, 90, 78],   # นักศึกษาที่ 1
    [72, 88, 95],   # นักศึกษาที่ 2
    [91, 76, 83],   # นักศึกษาที่ 3
]
names = ["สมชาย", "สมหญิง", "สมศักดิ์"]

for i, (name, scores) in enumerate(zip(names, student_scores)):
    avg = sum(scores) / len(scores)
    print(f"{name}: {scores} เฉลี่ย = {avg:.1f}")

🔍 9. การค้นหาใน List

Python 
fruits = ["แอปเปิล", "กล้วย", "ส้ม", "มะม่วง", "กล้วย"]

# ตรวจสอบว่ามีอยู่ใน List หรือไม่ (in operator)
print("กล้วย" in fruits)      # True
print("ลิ้นจี่" in fruits)    # False
print("ส้ม" not in fruits)    # False

# หา index ของค่า
print(fruits.index("ส้ม"))    # 2
print(fruits.index("กล้วย"))  # 1 (หาตัวแรกที่พบ)

# นับจำนวนครั้ง
print(fruits.count("กล้วย")) # 2

# ค้นหาด้วย loop
search = "มะม่วง"
for i, fruit in enumerate(fruits):
    if fruit == search:
        print(f"พบ '{search}' ที่ index {i}")

📊 10. ฟังก์ชันที่ใช้กับ List

Python 
scores = [85, 92, 78, 96, 88, 74, 91]

print(f"จำนวนสมาชิก:  len()    = {len(scores)}")
print(f"ค่าสูงสุด:    max()    = {max(scores)}")
print(f"ค่าต่ำสุด:    min()    = {min(scores)}")
print(f"ผลรวม:        sum()    = {sum(scores)}")
print(f"เฉลี่ย:                = {sum(scores)/len(scores):.2f}")
print(f"เรียงลำดับ:   sorted() = {sorted(scores)}")    # ไม่แก้ต้นฉบับ
print(f"กลับลำดับ:   sorted(reverse=True) = {sorted(scores, reverse=True)}")
print(f"ต้นฉบับยังคงเดิม: {scores}")
