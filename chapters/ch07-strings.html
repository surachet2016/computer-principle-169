บทที่ 7

String

เรียนรู้การทำงานกับข้อความใน Python ตั้งแต่การสร้าง String การเข้าถึงอักขระ การใช้ Method ต่าง ๆ ไปจนถึงการจัดรูปแบบข้อความ

CLO3: สามารถใช้งาน String และ Method ที่เกี่ยวข้องในการแก้ปัญหาได้

🔤 1. String คืออะไร?

String คือชนิดข้อมูลสำหรับเก็บข้อความใน Python โดยข้อความจะถูกล้อมรอบด้วยเครื่องหมายคำพูด สามารถใช้ได้ทั้ง single quote ('), double quote (") และ triple quote (''' หรือ """)

สิ่งสำคัญที่ต้องจำ: String ใน Python เป็น Immutable หมายความว่าเมื่อสร้างแล้วไม่สามารถเปลี่ยนแปลงอักขระภายในได้โดยตรง ต้องสร้าง String ใหม่เสมอ

วิธีสร้าง String
  • 'Hello' — single quote: เหมาะสำหรับข้อความทั่วไป
  • "World" — double quote: เหมาะเมื่อข้อความมี apostrophe เช่น "It's Python"
  • '''หลายบรรทัด''' — triple quote: รองรับข้อความหลายบรรทัด (multiline string)
Python 
# การสร้าง String แบบต่าง ๆ
s1 = 'สวัสดี'
s2 = "Python"
s3 = """นี่คือ
String หลายบรรทัด
ใช้ triple quote"""

print(s1)
print(s2)
print(s3)

# String เป็น Immutable
name = "Python"
# name[0] = "J"  # จะเกิด TypeError!
name = "Jython"  # ต้องสร้างใหม่
print(name)

# ความยาวของ String
print(len("Hello, World!"))  # 13

🔍 2. Indexing และ Slicing

การเข้าถึงอักขระใน String ทำได้เหมือนกับ List โดยใช้ Index ที่เริ่มจาก 0 และรองรับ Negative Index สำหรับนับจากท้าย

Slicing ใช้ไวยากรณ์ s[start:stop:step] เพื่อดึงส่วนของ String ออกมา

ตาราง Index ของ String "Python"
อักขระ Python
Positive Index 012345
Negative Index -6-5-4-3-2-1
Python 
s = "Python"

# Indexing
print(s[0])    # P — ตัวแรก
print(s[-1])   # n — ตัวสุดท้าย
print(s[2])    # t

# Slicing
print(s[1:4])  # yth — index 1 ถึง 3
print(s[:3])   # Pyt — ตั้งแต่ต้นถึง index 2
print(s[3:])   # hon — ตั้งแต่ index 3 จนจบ
print(s[::2])  # Pto — ข้ามทีละ 2
print(s[::-1]) # nohtyP — กลับด้าน String
เคล็ดลับ: การกลับด้าน String

ใช้ s[::-1] เพื่อกลับด้าน String ได้ทันที เป็นเทคนิคที่นิยมใช้ตรวจสอบ Palindrome (คำที่อ่านหน้าหลังเหมือนกัน)

🔧 3. String Methods

Python มี Built-in Methods สำหรับ String จำนวนมาก ช่วยให้จัดการข้อความได้สะดวก เรียกใช้ด้วยไวยากรณ์ string.method()

ตารางสรุป String Methods ที่ใช้บ่อย
Methodคำอธิบายตัวอย่าง
upper()แปลงเป็นตัวพิมพ์ใหญ่"hello".upper() → "HELLO"
lower()แปลงเป็นตัวพิมพ์เล็ก"HELLO".lower() → "hello"
strip()ลบช่องว่างหน้า-หลัง" hi ".strip() → "hi"
split(sep)แยก String เป็น List"a,b,c".split(",") → ["a","b","c"]
join(iterable)รวม List เป็น String"-".join(["a","b"]) → "a-b"
replace(old, new)แทนที่ข้อความ"abc".replace("b","X") → "aXc"
find(sub)หาตำแหน่ง (-1 ถ้าไม่พบ)"hello".find("ll") → 2
count(sub)นับจำนวนที่ปรากฏ"banana".count("a") → 3
startswith(prefix)ตรวจสอบตัวขึ้นต้น"Hello".startswith("He") → True
endswith(suffix)ตรวจสอบตัวลงท้าย"Hello".endswith("lo") → True
Python 
text = "  Hello, Python World!  "

print(text.strip())            # ลบช่องว่าง
print(text.strip().upper())    # ตัวพิมพ์ใหญ่
print(text.strip().lower())    # ตัวพิมพ์เล็ก

s = "apple,banana,cherry"
fruits = s.split(",")          # แยกด้วย ,
print(fruits)                  # ['apple', 'banana', 'cherry']
print(" | ".join(fruits))      # รวมด้วย |

sentence = "the quick brown fox"
print(sentence.replace("fox", "cat"))  # แทนที่คำ
print(sentence.find("quick"))          # หาตำแหน่ง -> 4
print(sentence.count("o"))             # นับตัว o -> 2
print(sentence.startswith("the"))      # True
print(sentence.endswith("dog"))        # False

✨ 4. การจัดรูปแบบ String (f-string & format)

Python มีหลายวิธีในการนำค่าตัวแปรแทรกลงใน String วิธีที่แนะนำในปัจจุบันคือ f-string (Python 3.6+) ซึ่งอ่านง่ายและมีประสิทธิภาพสูง

วิธีการจัดรูปแบบ String
  • f-string: f"Hello {name}" — วิธีที่แนะนำ
  • .format(): "Hello {}".format(name) — รองรับ Python เก่า
  • % operator: "Hello %s" % name — รูปแบบเก่า (ไม่แนะนำ)
Python 
name = "สมชาย"
age = 20
score = 88.567

# f-string (แนะนำ)
print(f"สวัสดี {name} อายุ {age} ปี")
print(f"คะแนน: {score:.2f}")        # ทศนิยม 2 ตำแหน่ง
print(f"คะแนน: {score:.0f}")        # ปัดเป็นจำนวนเต็ม
print(f"{'Python':>10}")             # จัดชิดขวา 10 ตำแหน่ง
print(f"{'Python':<10}|")           # จัดชิดซ้าย 10 ตำแหน่ง
print(f"{'Python':^10}")             # จัดกึ่งกลาง 10 ตำแหน่ง

# .format()
print("สวัสดี {} อายุ {} ปี".format(name, age))
print("ผล: {score:.2f}".format(score=score))

# คำนวณใน f-string ได้เลย
print(f"10 + 5 = {10 + 5}")
print(f"ชื่อตัวพิมพ์ใหญ่: {name.upper()}")
Format Specifiers ที่ใช้บ่อย
รูปแบบความหมายตัวอย่าง
:.2fทศนิยม 2 ตำแหน่งf"{3.14159:.2f}" → 3.14
:dจำนวนเต็มf"{42:d}" → 42
:,ใส่ comma คั่นพันf"{1000000:,}" → 1,000,000
:10sความกว้าง 10 ตัวf"{'hi':10s}" → "hi "

🔗 5. การเชื่อมต่อและทำซ้ำ

String สามารถนำมาต่อกัน (Concatenation) ด้วยเครื่องหมาย + และทำซ้ำด้วย * นอกจากนี้ยังมีตัวดำเนินการ in ใช้ตรวจสอบว่ามีอักขระหรือข้อความย่อยอยู่ใน String หรือไม่

Python 
# Concatenation — เชื่อมต่อ String
first = "Hello"
second = "World"
result = first + ", " + second + "!"
print(result)  # Hello, World!

# Repetition — ทำซ้ำ String
line = "-" * 30
print(line)            # พิมพ์ --- 30 ตัว
print("Ha" * 3)        # HaHaHa

# in operator — ตรวจสอบการมีอยู่
text = "Python is awesome"
print("Python" in text)    # True
print("Java" in text)      # False
print("is" not in text)    # False

# วนลูปผ่านอักขระใน String
for ch in "Hello":
    print(ch, end=" ")     # H e l l o
