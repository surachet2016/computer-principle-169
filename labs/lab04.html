Conditions

ฝึกใช้คำสั่งเงื่อนไข if, if-else, if-elif-else และ nested if เพื่อควบคุมการทำงานของโปรแกรม

CLO1: เขียนโปรแกรมที่มีการตัดสินใจได้ CLO3: แก้ปัญหาด้วยการออกแบบเงื่อนไขได้

🎯 วัตถุประสงค์

นักศึกษาจะสามารถ...
  • เขียน if statement พร้อมเข้าใจความสำคัญของ indentation ได้
  • ใช้ if-else เลือกทำงานสองทางตามเงื่อนไขได้
  • ใช้ if-elif-else จัดการเงื่อนไขหลายกรณีได้
  • เขียน nested if เพื่อตรวจสอบเงื่อนไขซ้อนกันได้

📋 ขั้นตอนที่ 1: if statement

คำสั่ง if เป็นคำสั่งพื้นฐานสำหรับการตัดสินใจ โค้ดที่อยู่ภายใน if จะทำงานก็ต่อเมื่อเงื่อนไขเป็น True เท่านั้น สิ่งสำคัญที่สุดในการเขียน Python คือ indentation (การเยื้องบรรทัด) ต้องใช้ช่องว่าง 4 ตัวอักษรอย่างสม่ำเสมอ ผิด indentation จะทำให้เกิด IndentationError

Python 
age = 20

# รูปแบบ: if เงื่อนไข: แล้วเยื้องโค้ด 4 ช่อง
if age >= 18:
    print("บรรลุนิติภาวะแล้ว")
    print("มีสิทธิ์เลือกตั้ง")

print("บรรทัดนี้ทำงานเสมอ")  # อยู่นอก if

📋 ขั้นตอนที่ 2: if-else

if-else ใช้เมื่อต้องการแยกสองเส้นทาง ถ้าเงื่อนไขเป็น True จะทำโค้ดใน if ถ้าเป็น False จะทำโค้ดใน else แทน ทั้งสองกรณีต้องเลือกทำอย่างหนึ่งเสมอ

Python 
number = 7

# ตรวจว่าเลขคี่หรือคู่
if number % 2 == 0:
    print(f"{number} เป็นเลขคู่")
else:
    print(f"{number} เป็นเลขคี่")

# อีกตัวอย่าง: ตรวจอุณหภูมิ
temp = 35
if temp > 30:
    print("อากาศร้อน ควรดื่มน้ำมาก ๆ")
else:
    print("อากาศพอสบาย")

📋 ขั้นตอนที่ 3: if-elif-else

เมื่อมีมากกว่า 2 กรณี ใช้ elif (else if) เพิ่มเงื่อนไขได้ไม่จำกัด Python จะตรวจสอบเงื่อนไขจากบนลงล่าง และจะหยุดที่เงื่อนไขแรกที่เป็น True เท่านั้น ส่วน else ที่อยู่ท้ายสุดจะทำงานเมื่อไม่มีเงื่อนไขใดเป็น True

Python 
score = 75

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"คะแนน {score} → เกรด {grade}")

📋 ขั้นตอนที่ 4: Nested if

Nested if คือการเขียน if ซ้อนอยู่ภายใน if อีกชั้น ใช้ตรวจสอบเงื่อนไขหลายระดับ แต่ต้องระวัง indentation ให้ถูกต้อง และไม่ควร nested ลึกเกินไปเพราะทำให้โค้ดอ่านยาก

Python 
# ระบบตรวจสอบสิทธิ์เข้าใช้งาน
is_member = True
age = 20

if is_member:
    if age >= 18:
        print("ยินดีต้อนรับ! เข้าใช้งานได้เต็มรูปแบบ")
    else:
        print("สมาชิก แต่อายุน้อยกว่า 18 ปี — เข้าได้บางส่วน")
else:
    print("กรุณาสมัครสมาชิกก่อน")
