Lab 8

Functions

ฝึกนิยามและเรียกใช้ฟังก์ชัน พารามิเตอร์ ค่าเริ่มต้น และการ return หลายค่า

CLO3: สามารถออกแบบและใช้ฟังก์ชันแก้ปัญหาได้

🎯 วัตถุประสงค์

นักศึกษาจะสามารถ...
  • นิยามฟังก์ชันด้วย def และเรียกใช้งานได้
  • ใช้ positional argument, keyword argument และ default value ได้
  • ใช้ return เพื่อส่งค่าหลายค่าออกจากฟังก์ชันได้
  • เรียกฟังก์ชันภายในฟังก์ชันอื่น (ฟังก์ชันซ้อนกัน) ได้

📋 ขั้นตอนที่ 1: นิยามและเรียกใช้ฟังก์ชัน

ฟังก์ชันคือบล็อกโค้ดที่มีชื่อ สามารถเรียกใช้ซ้ำได้ นิยามด้วยคำสั่ง def ตามด้วยชื่อและวงเล็บ () เนื้อหาของฟังก์ชันต้องเยื้องเข้า (indent) และใช้ return เพื่อส่งค่ากลับ ถ้าไม่มี return ฟังก์ชันจะคืนค่า None

Python 
def greet(name):
    """ทักทายผู้ใช้"""
    message = f"สวัสดี, {name}!"
    return message

# เรียกใช้ฟังก์ชัน
result = greet("สมชาย")
print(result)       # สวัสดี, สมชาย!
print(greet("ครู")) # สวัสดี, ครู!

📋 ขั้นตอนที่ 2: Parameters vs Arguments

Parameter คือตัวแปรในวงเล็บของ def ส่วน Argument คือค่าที่ส่งเข้าฟังก์ชันเมื่อเรียกใช้ สามารถส่งแบบ positional (ตามลำดับ) หรือ keyword (ระบุชื่อ) ก็ได้ นอกจากนี้ยังกำหนดค่าเริ่มต้น (default value) ให้ parameter ได้ ซึ่งจะใช้เมื่อไม่ได้ส่ง argument นั้นมา

Python 
def introduce(name, age, lang="Python"):
    print(f"ฉันชื่อ {name} อายุ {age} ปี เขียน {lang}")

# positional
introduce("สมชาย", 20)           # ใช้ default lang

# keyword
introduce(age=22, name="สมหญิง")  # สลับลำดับได้

# ส่งครบทุก argument
introduce("สมศักดิ์", 21, "Java")

📋 ขั้นตอนที่ 3: Return หลายค่า

Python ช่วยให้ฟังก์ชัน return หลายค่าพร้อมกันได้ด้วย tuple โดยเขียน return a, b, c ฝั่งผู้เรียกสามารถ unpack ค่าออกมาได้เลย เช่น x, y, z = func() หรือรับเป็น tuple ก้อนเดียว

Python 
def min_max_avg(numbers):
    """คืน ค่าต่ำสุด สูงสุด และเฉลี่ย"""
    return min(numbers), max(numbers), sum(numbers)/len(numbers)

data = [4, 7, 2, 9, 1, 8]
low, high, avg = min_max_avg(data)

print(f"ต่ำสุด: {low}")
print(f"สูงสุด: {high}")
print(f"เฉลี่ย: {avg:.2f}")

📋 ขั้นตอนที่ 4: ฟังก์ชันซ้อนกัน

ฟังก์ชันสามารถเรียกฟังก์ชันอื่นภายในตัวเองได้ (function composition) ซึ่งช่วยให้โค้ดแบ่งหน้าที่ชัดเจนและนำกลับมาใช้ซ้ำได้ง่าย ตัวอย่างเช่น ฟังก์ชันแสดงรายงานที่เรียกฟังก์ชันคำนวณเกรดและสถิติอีกทีหนึ่ง

Python 
def get_grade(score):
    if score >= 80: return "A"
    elif score >= 70: return "B"
    elif score >= 60: return "C"
    else: return "F"

def show_result(name, score):
    grade = get_grade(score)   # เรียกฟังก์ชันอีกตัว
    print(f"{name}: {score} คะแนน → เกรด {grade}")

show_result("สมชาย", 85)
show_result("สมหญิง", 65)
