Data Types & Type Conversion

เรียนรู้ชนิดข้อมูลพื้นฐานใน Python และการแปลงระหว่างชนิดข้อมูล

CLO1: เข้าใจและใช้งานชนิดข้อมูลพื้นฐานได้

🎯 วัตถุประสงค์

นักศึกษาจะสามารถ...
  • บอกชนิดข้อมูลพื้นฐานของ Python ได้ ได้แก่ int, float, str, bool, None
  • ใช้ฟังก์ชัน type() ตรวจสอบชนิดข้อมูลได้
  • แปลงชนิดข้อมูลด้วย int(), float(), str(), bool() ได้
  • รู้จักข้อผิดพลาดที่เกิดจากการแปลงชนิดข้อมูลที่ไม่ถูกต้อง

📋 ขั้นตอนที่ 1: ชนิดข้อมูลพื้นฐาน

Python มีชนิดข้อมูลพื้นฐาน 5 ชนิดหลัก ได้แก่ int (จำนวนเต็ม) เช่น 42, -10 — float (จำนวนทศนิยม) เช่น 3.14, -0.5 — str (ข้อความ) เช่น "สวัสดี" — bool (ค่าความจริง) คือ True หรือ False เท่านั้น — และ None (ค่าว่าง ไม่มีค่า) ใช้ฟังก์ชัน type() เพื่อตรวจสอบชนิดของตัวแปรได้

Python 
# ชนิดข้อมูลพื้นฐาน
a = 42          # int
b = 3.14        # float
c = "สวัสดี"    # str
d = True        # bool
e = None        # NoneType

# ตรวจสอบชนิดข้อมูลด้วย type()
print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'str'>
print(type(d))  # <class 'bool'>
print(type(e))  # <class 'NoneType'>

# แสดงเฉพาะชื่อชนิด
print(type(a).__name__)  # int

📋 ขั้นตอนที่ 2: Type Conversion

บางครั้งเราต้องการแปลงข้อมูลจากชนิดหนึ่งไปเป็นอีกชนิดหนึ่ง เรียกว่า Type Conversion Python มีฟังก์ชันสำหรับแปลงชนิดข้อมูลหลัก ได้แก่ int() แปลงเป็นจำนวนเต็ม, float() แปลงเป็นทศนิยม, str() แปลงเป็นข้อความ และ bool() แปลงเป็นค่าความจริง ข้อควรระวัง: int(3.99) จะได้ 3 ไม่ใช่ 4 เพราะ Python ตัดทศนิยมทิ้ง ไม่ปัดขึ้น

Python 
# การแปลงชนิดข้อมูล (Type Conversion)
x = 42
y = 3.99
z = "100"

# int() — แปลงเป็นจำนวนเต็ม (ตัดทศนิยมทิ้ง ไม่ปัดขึ้น)
print(int(y))         # 3 (ไม่ใช่ 4!)
print(int(z))         # 100

# float() — แปลงเป็นทศนิยม
print(float(x))       # 42.0
print(float("3.14"))  # 3.14

# str() — แปลงเป็นข้อความ
print(str(x))         # "42"
print(str(y))         # "3.99"

# bool() — 0, "" (ว่าง), None คือ False ส่วนที่เหลือคือ True
print(bool(0))        # False
print(bool(1))        # True
print(bool(""))       # False
print(bool("Hi"))     # True

📋 ขั้นตอนที่ 3: ข้อควรระวัง

การแปลงชนิดข้อมูลบางกรณีทำไม่ได้และจะเกิด ValueError ขึ้น เช่น int("abc") เพราะ "abc" ไม่ใช่ตัวเลข และ int("3.14") ก็ทำไม่ได้โดยตรง ต้องแปลงเป็น float ก่อน เราสามารถใช้ try/except จัดการ error ได้อย่างสวยงามแทนที่จะปล่อยให้โปรแกรมหยุดทำงาน

Python 
# กรณีที่ทำได้ vs ทำไม่ได้
print(float("3.14"))           # ได้: 3.14
print(int(float("3.14")))      # ได้: 3 (แปลงสองขั้น)

# จัดการ error ด้วย try/except
try:
    result = int("abc")
    print(result)
except ValueError as e:
    print(f"เกิดข้อผิดพลาด: {e}")

# ทดสอบอีกกรณี
try:
    result = int("3.14")       # ทำไม่ได้โดยตรง
except ValueError:
    print("int('3.14') ทำไม่ได้ — ต้องใช้ float() ก่อน")
    print("วิธีถูกต้อง:", int(float("3.14")))
