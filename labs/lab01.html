Lab 1

Hello World & Variables

เริ่มต้นเขียนโปรแกรม Python ด้วยคำสั่ง print และการใช้ตัวแปรพื้นฐาน

CLO1: เขียนและอ่านโปรแกรม Python พื้นฐานได้

🎯 วัตถุประสงค์

นักศึกษาจะสามารถ...
  • ใช้คำสั่ง print() แสดงผลข้อมูลออกทางหน้าจอได้
  • ประกาศและใช้งานตัวแปร (variable) ชนิดต่าง ๆ ได้
  • เขียน comment เพื่ออธิบายโค้ดได้ถูกต้อง
  • เขียนโปรแกรมแนะนำตัวอย่างง่ายได้

📋 ขั้นตอนที่ 1: Hello World

คำสั่งแรกที่นักเขียนโปรแกรมทุกคนเริ่มต้นคือการแสดงคำว่า "Hello, World!" ใน Python ใช้ฟังก์ชัน print() ซึ่งสามารถรับ argument ได้หลายตัว คั่นด้วยเครื่องหมายจุลภาค Python จะเว้นช่องว่างระหว่าง argument โดยอัตโนมัติ นอกจากนี้ยังสามารถกำหนด sep (ตัวคั่น) และ end (ตัวอักษรสุดท้าย) ได้อีกด้วย

Python 
# แสดงผลพื้นฐาน
print("Hello, World!")

# print หลาย argument — Python ใส่ช่องว่างให้อัตโนมัติ
print("ชื่อ:", "สมชาย", "อายุ:", 20)

# กำหนด sep (ตัวคั่น) และ end (ตัวอักษรสุดท้าย)
print("Python", "is", "fun", sep="-")
print("บรรทัดนี้", end=" ")
print("ต่อกันบรรทัดเดียว")

📋 ขั้นตอนที่ 2: ตัวแปร (Variables)

ตัวแปรใน Python ไม่ต้องประกาศชนิดข้อมูลล่วงหน้า แค่กำหนดค่าได้เลย Python จะกำหนดชนิดข้อมูลให้อัตโนมัติตามค่าที่กำหนด ชื่อตัวแปรควรสื่อความหมาย เริ่มต้นด้วยตัวอักษรหรือ underscore (_) และไม่ใช้ชื่อที่ Python ใช้งานอยู่แล้ว เช่น print, if, for

Python 
# ประกาศตัวแปรชนิดต่าง ๆ
name = "สมชาย ใจดี"        # ข้อความ (str)
age = 20                    # จำนวนเต็ม (int)
gpa = 3.75                  # จำนวนทศนิยม (float)
faculty = "วิศวกรรมศาสตร์"  # ข้อความ (str)
student_id = "6500001"      # ข้อความ (str)

# แสดงผลตัวแปร
print("ชื่อ:", name)
print("อายุ:", age, "ปี")
print("GPA:", gpa)
print("คณะ:", faculty)

📋 ขั้นตอนที่ 3: Comments

Comment คือข้อความที่ Python จะไม่นำไปประมวลผล ใช้เพื่ออธิบายโค้ดให้เข้าใจง่าย มี 2 รูปแบบหลัก คือ Single-line comment ใช้เครื่องหมาย # นำหน้า และ Multi-line comment ใช้ triple quotes """...""" ครอบข้อความ การเขียน comment ที่ดีเป็นนิสัยสำคัญของโปรแกรมเมอร์

Python 
# นี่คือ single-line comment
# ทุกอย่างหลัง # จะถูก Python ข้าม

x = 10  # comment ต่อท้ายโค้ดก็ได้

"""
นี่คือ multi-line comment หรือ docstring
ใช้เครื่องหมาย triple quotes สามตัวครอบข้อความ
สามารถขึ้นบรรทัดใหม่ได้ตามต้องการ
"""

print("โค้ดบรรทัดนี้ทำงานปกติ")
print("x =", x)
