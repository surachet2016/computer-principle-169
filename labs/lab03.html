📋 ขั้นตอนที่ 1: ตัวดำเนินการทางคณิตศาสตร์

Python มีตัวดำเนินการทางคณิตศาสตร์ครบถ้วน ที่ต้องระวังเป็นพิเศษคือความต่างระหว่าง / (หารได้ทศนิยม) กับ // (หารปัดทิ้งทศนิยม) และ % (เศษจากการหาร) ซึ่งนิยมใช้ตรวจเลขคี่/คู่ ส่วน ** คือยกกำลัง

Python 
a = 17
b = 5

print(a + b)   # บวก → 22
print(a - b)   # ลบ → 12
print(a * b)   # คูณ → 85
print(a / b)   # หาร → 3.4 (ได้ทศนิยม)
print(a // b)  # หารปัดทิ้ง → 3
print(a % b)   # เศษ → 2
print(a ** 2)  # ยกกำลัง → 289

📋 ขั้นตอนที่ 2: ตัวดำเนินการเปรียบเทียบ

ตัวดำเนินการเปรียบเทียบ ได้แก่ == (เท่ากัน), != (ไม่เท่ากัน), <, >, <=, >= ผลลัพธ์จะเป็น bool เสมอ คือ True หรือ False เท่านั้น ตัวดำเนินการเหล่านี้มักใช้ร่วมกับ if statement

Python 
x = 10
y = 20

print(x == y)   # False — ไม่เท่ากัน
print(x != y)   # True  — ไม่เท่ากัน
print(x < y)    # True  — น้อยกว่า
print(x > y)    # False — ไม่มากกว่า
print(x <= 10)  # True  — น้อยกว่าหรือเท่ากับ
print(x >= 10)  # True  — มากกว่าหรือเท่ากับ

📋 ขั้นตอนที่ 3: ตัวดำเนินการตรรกศาสตร์

ตัวดำเนินการตรรกศาสตร์ใช้รวมเงื่อนไขหลายข้อเข้าด้วยกัน ได้แก่ and (และ — ทั้งสองต้องจริง), or (หรือ — อย่างน้อยหนึ่งต้องจริง) และ not (ไม่ — กลับค่า) นิยมใช้กับ if เพื่อตรวจสอบหลายเงื่อนไขพร้อมกัน

Python 
age = 20
has_id = True

# and — ทั้งสองเงื่อนไขต้องเป็น True
print(age >= 18 and has_id)   # True

# or — อย่างน้อยหนึ่งเงื่อนไขเป็น True
print(age >= 18 or has_id)    # True
print(age < 18 or has_id)     # True

# not — กลับค่า
print(not has_id)             # False
print(not (age < 18))         # True

# Truth table สรุป
print(True and False)   # False
print(True or False)    # True
print(not True)         # False

📋 ขั้นตอนที่ 4: การแสดงผลด้วย f-string

f-string (Formatted String Literals) เป็นวิธีที่สะดวกที่สุดในการแทรกค่าตัวแปรลงในข้อความ เริ่มต้น string ด้วย f"..." แล้วใส่ตัวแปรในวงเล็บปีกกา {} นอกจากนี้ยังกำหนดรูปแบบได้ เช่น {value:.2f} แสดงทศนิยม 2 ตำแหน่ง และ {name:>10} จัดชิดขวาในความกว้าง 10 ตัวอักษร

Python 
name = "สมชาย"
score = 87.3456
pi = 3.14159265

# แทรกตัวแปรพื้นฐาน
print(f"ชื่อ: {name}")

# กำหนดจำนวนทศนิยม
print(f"คะแนน: {score:.2f}")   # 87.35
print(f"Pi: {pi:.4f}")         # 3.1416

# จัดตำแหน่งข้อความ
print(f"{'รายการ':<10} {'ราคา':>8}")
print(f"{'ข้าวผัด':<10} {45:>8.2f}")
print(f"{'ต้มยำ':<10} {120:>8.2f}")
