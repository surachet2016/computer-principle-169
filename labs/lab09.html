Lab 9

File Handling

ฝึกเปิด เขียน และอ่านไฟล์ด้วย with statement และจัดการข้อผิดพลาดด้วย try/except

CLO3: สามารถเขียนโปรแกรมอ่านเขียนไฟล์ได้

🎯 วัตถุประสงค์

นักศึกษาจะสามารถ...
  • เปิดและปิดไฟล์ด้วย open() และระบุ mode (r/w/a) ได้อย่างถูกต้อง
  • เขียนข้อมูลลงไฟล์ด้วย write() และ writelines() ได้
  • อ่านข้อมูลจากไฟล์ด้วย read(), readline(), readlines() และ for loop ได้
  • ใช้ with statement เพื่อจัดการไฟล์อย่างปลอดภัยได้
  • ดักจับข้อผิดพลาด FileNotFoundError ด้วย try/except ได้

📋 ขั้นตอนที่ 1: การเปิดและปิดไฟล์

ฟังก์ชัน open(filename, mode) ใช้เปิดไฟล์ โดย mode มีค่าหลักดังนี้: "r" อ่านอย่างเดียว (ต้องมีไฟล์อยู่แล้ว), "w" เขียนใหม่ (ล้างเนื้อหาเดิม), "a" ต่อท้าย (append) และ "r+" อ่านและเขียน หลังใช้งานต้องปิดไฟล์ด้วย close() เสมอ

Python 
# เปิดแบบ manual — ต้องปิดเอง
f = open("/tmp/demo.txt", "w", encoding="utf-8")
f.write("สวัสดี Python!\n")
f.close()   # อย่าลืมปิด!

# mode ที่ใช้บ่อย
# "r"  → อ่านอย่างเดียว
# "w"  → เขียนใหม่ (ลบของเดิม)
# "a"  → ต่อท้าย
# "rb" → อ่าน binary

📋 ขั้นตอนที่ 2: การเขียนไฟล์

write(string) เขียน string ลงไฟล์โดยไม่ขึ้นบรรทัดใหม่อัตโนมัติ ต้องเติม \n เอง ส่วน writelines(list) เขียน list ของ string ทีละรายการ ก็ต้องเติม \n ในแต่ละรายการด้วยเช่นกัน

Python 
lines = ["แอปเปิล\n", "กล้วย\n", "ส้ม\n"]

f = open("/tmp/fruits.txt", "w", encoding="utf-8")
f.write("รายการผลไม้\n")
f.write("=" * 20 + "\n")
f.writelines(lines)       # เขียนทั้ง list
f.close()

print("เขียนไฟล์เสร็จแล้ว")

📋 ขั้นตอนที่ 3: การอ่านไฟล์

มีวิธีอ่านไฟล์หลายแบบ: read() อ่านทั้งหมดเป็น string เดียว, readline() อ่านทีละบรรทัด, readlines() อ่านทั้งหมดเป็น list ของ string, และการวนซ้ำด้วย for line in f ซึ่งประหยัดหน่วยความจำที่สุดสำหรับไฟล์ขนาดใหญ่

Python 
f = open("/tmp/fruits.txt", "r", encoding="utf-8")

# อ่านทั้งหมด
content = f.read()
print(content)
f.close()

# อ่านทีละบรรทัด (ประหยัดหน่วยความจำ)
f = open("/tmp/fruits.txt", "r", encoding="utf-8")
for line in f:
    print(line.strip())   # ตัด \n ออก
f.close()

📋 ขั้นตอนที่ 4: with Statement

with open(...) as f: คือวิธีที่แนะนำให้ใช้ (best practice) เพราะ Python จะปิดไฟล์ให้อัตโนมัติเมื่อออกจาก block ไม่ว่าจะเกิดข้อผิดพลาดหรือไม่ก็ตาม ช่วยป้องกันปัญหาไฟล์ค้างเปิด

Python 
# เขียนไฟล์ด้วย with
with open("/tmp/note.txt", "w", encoding="utf-8") as f:
    f.write("บันทึกด้วย with statement\n")
# ไฟล์ปิดอัตโนมัติที่นี่

# อ่านไฟล์ด้วย with
with open("/tmp/note.txt", "r", encoding="utf-8") as f:
    for i, line in enumerate(f, 1):
        print(f"{i}: {line.strip()}")

📋 ขั้นตอนที่ 5: try/except — FileNotFoundError

ถ้าพยายามอ่านไฟล์ที่ไม่มีอยู่จริง Python จะ raise FileNotFoundError ทำให้โปรแกรมหยุดทำงาน การใช้ try/except ดักจับข้อผิดพลาดนี้ทำให้โปรแกรมแสดงข้อความเตือนและทำงานต่อได้อย่างราบรื่น

Python 
filename = "/tmp/ไม่มีไฟล์นี้.txt"

try:
    with open(filename, "r", encoding="utf-8") as f:
        content = f.read()
        print(content)
except FileNotFoundError:
    print(f"ไม่พบไฟล์: {filename}")
except PermissionError:
    print("ไม่มีสิทธิ์เข้าถึงไฟล์นี้")
finally:
    print("จบการทำงาน")
