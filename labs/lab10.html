Mini Project

รวมทักษะทั้งหมด: List, String, ฟังก์ชัน และไฟล์ เพื่อสร้างระบบบันทึกคะแนนนักศึกษา

CLO2: วิเคราะห์และออกแบบโปรแกรมจากโจทย์ได้ CLO3: เขียนโปรแกรมครบวงจรด้วย Python ได้ CLO4: ทดสอบและแก้ไขโปรแกรมได้

📌 โจทย์โครงงาน

ระบบบันทึกคะแนนนักศึกษา (Student Score System)

สร้างโปรแกรม Python ที่ทำงานได้ครบตามข้อกำหนดต่อไปนี้:

  • เก็บชื่อและคะแนนนักศึกษาใน List
  • มี ฟังก์ชัน หาค่าเฉลี่ย ค่าสูงสุด และค่าต่ำสุด
  • ตัดเกรด (A/B/C/D/F) และตรวจสอบสถานะ ผ่าน/ไม่ผ่าน
  • แสดง รายงาน แบบตารางที่อ่านง่าย
  • บันทึกผล ลงไฟล์ /tmp/report.txt

🗺️ ผังงานโปรแกรม

โปรแกรมทำงานตามลำดับขั้นตอนดังนี้:

  │   เริ่มต้น   │
  └──────┬──────┘
         │
         ▼
  ┌──────────────────────────┐
  │  กำหนดข้อมูลนักศึกษา    │
  │  (students[], scores[])  │
  └──────────┬───────────────┘
             │
             ▼
  ┌──────────────────────────┐

  1. อธิบายการทำงานของโปรแกรมในภาษาของตัวเอง (5-10 ประโยค) โดยครอบคลุมหน้าที่ของแต่ละฟังก์ชัน
