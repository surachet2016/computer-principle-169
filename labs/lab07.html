Strings

ฝึกใช้ String: การ index, slice, เมธอดต่าง ๆ และ f-string formatting

CLO3: สามารถประมวลผลข้อความด้วย String methods ได้

🎯 วัตถุประสงค์

นักศึกษาจะสามารถ...
  • เข้าถึงตัวอักษรใน String ด้วย index และตัดส่วนด้วย slicing ได้
  • ใช้เมธอด upper(), lower(), strip(), split(), join(), replace() ได้
  • ตรวจสอบเนื้อหาของ String ด้วย startswith(), endswith(), in, find(), count() ได้
  • จัดรูปแบบผลลัพธ์ด้วย f-string ได้

📋 ขั้นตอนที่ 1: Indexing และ Slicing

String ใน Python ทำงานคล้าย List ของตัวอักษร สามารถเข้าถึงทีละตัวด้วย index โดย s[0] คือตัวแรก และ s[-1] คือตัวสุดท้าย การ slicing ด้วย s[start:end] จะตัดส่วนของ String ออกมา และ s[::-1] จะกลับลำดับตัวอักษรทั้งหมด

Python 
s = "Hello, Python!"

print(s[0])      # H  (ตัวแรก)
print(s[-1])     # !  (ตัวสุดท้าย)
print(s[7:13])   # Python  (slice)
print(s[:5])     # Hello
print(s[::-1])   # !nohtyP ,olleH  (กลับด้าน)

📋 ขั้นตอนที่ 2: String Methods ที่ใช้บ่อย

Python มีเมธอดสำหรับจัดการ String มากมาย ที่ใช้บ่อยได้แก่ upper() / lower() แปลงตัวพิมพ์, strip() ตัดช่องว่างหัวท้าย, split(sep) แยก String เป็น List, join(list) รวม List เป็น String, และ replace(old, new) แทนที่ข้อความ

Python 
text = "  Hello World  "

print(text.strip())          # "Hello World"
print(text.strip().upper())  # "HELLO WORLD"
print(text.strip().lower())  # "hello world"

words = text.strip().split()      # ['Hello', 'World']
joined = "-".join(words)           # "Hello-World"
print(joined)

new_text = text.strip().replace("World", "Python")
print(new_text)  # "Hello Python"

📋 ขั้นตอนที่ 3: การตรวจสอบ String

Python มีเมธอดและตัวดำเนินการสำหรับตรวจสอบเนื้อหาของ String เช่น startswith(prefix) และ endswith(suffix) ตรวจสอบจุดเริ่มและจุดสิ้นสุด, in ตรวจว่า substring อยู่ใน String หรือไม่, find(sub) คืน index ที่พบ (คืน -1 ถ้าไม่พบ), และ count(sub) นับจำนวนครั้งที่ปรากฏ

Python 
email = "student@pnu.ac.th"

print(email.startswith("student"))    # True
print(email.endswith(".th"))          # True
print("pnu" in email)                 # True
print(email.find("@"))               # 7  (index ของ @)
print(email.count("."))              # 2  (จำนวน .)

📋 ขั้นตอนที่ 4: f-string Formatting

f-string (Formatted String Literals) คือวิธีแทรกค่าตัวแปรลงใน String อย่างกระชับ ใช้ f"..." และใส่ตัวแปรหรือนิพจน์ในวงเล็บปีกกา {} สามารถกำหนดรูปแบบตัวเลขได้ด้วย format spec เช่น {x:.2f} ทศนิยม 2 ตำแหน่ง, {n:05d} เติมศูนย์ให้ครบ 5 หลัก, และ {s:<10} จัดชิดซ้ายใน 10 ช่อง

Python 
name  = "สมชาย"
score = 87.5
num   = 42

print(f"ชื่อ: {name}")
print(f"คะแนน: {score:.2f}")     # 87.50
print(f"รหัส: {num:05d}")        # 00042
print(f"{'ซ้าย':<10}|{'ขวา':>10}")   # จัดตำแหน่ง
