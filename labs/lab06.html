Lists

ฝึกใช้ List ในการเก็บข้อมูลหลายค่า การเข้าถึง การแก้ไข และ List Comprehension

CLO3: สามารถใช้ List จัดการชุดข้อมูลในโปรแกรมได้

🎯 วัตถุประสงค์

นักศึกษาจะสามารถ...
  • สร้าง List และเข้าถึงสมาชิกด้วย index บวกและลบได้
  • ใช้เมธอด append(), insert(), remove(), pop(), sort(), reverse() ได้
  • วนซ้ำผ่าน List ด้วย for loop และ enumerate() ได้
  • เขียน List Comprehension เพื่อสร้าง List ใหม่อย่างกระชับได้

📋 ขั้นตอนที่ 1: การสร้างและเข้าถึง List

List คือโครงสร้างข้อมูลที่เก็บหลายค่าในตัวแปรเดียว สร้างด้วยวงเล็บเหลี่ยม [] และคั่นด้วยเครื่องหมายจุลภาค การเข้าถึงสมาชิกใช้ index ซึ่งเริ่มจาก 0 หรือใช้ index ลบเพื่อนับจากท้าย เช่น -1 คือสมาชิกสุดท้าย ฟังก์ชัน len() ใช้หาความยาวของ List

Python 
fruits = ["แอปเปิล", "กล้วย", "ส้ม", "มะม่วง"]

print(fruits[0])    # แอปเปิล  (index แรก)
print(fruits[-1])   # มะม่วง  (index สุดท้าย)
print(fruits[1:3])  # ['กล้วย', 'ส้ม']  (slicing)
print(len(fruits))  # 4

📋 ขั้นตอนที่ 2: การแก้ไข List

List สามารถแก้ไขได้หลังสร้าง (mutable) โดยใช้เมธอดต่าง ๆ เช่น append(x) เพิ่มท้าย List, insert(i, x) แทรกที่ตำแหน่ง i, remove(x) ลบค่า x ตัวแรก, pop(i) ลบและคืนค่าที่ตำแหน่ง i (ถ้าไม่ระบุจะลบท้าย), sort() เรียงลำดับ, และ reverse() กลับด้าน

Python 
nums = [3, 1, 4, 1, 5, 9, 2, 6]

nums.append(7)        # เพิ่ม 7 ท้าย List
nums.insert(0, 0)     # แทรก 0 ที่ตำแหน่งแรก
nums.remove(1)        # ลบ 1 ตัวแรก
last = nums.pop()     # ลบและรับค่าท้ายสุด

nums.sort()           # เรียงจากน้อยไปมาก
print(nums)

nums.sort(reverse=True)   # เรียงจากมากไปน้อย
print(nums)

📋 ขั้นตอนที่ 3: วนซ้ำ List

การวนซ้ำผ่าน List ทำได้ด้วย for loop โดยตรง หรือใช้ enumerate() เพื่อให้ได้ทั้ง index และค่าในเวลาเดียวกัน ซึ่งมีประโยชน์เมื่อต้องการทราบลำดับของสมาชิก

Python 
colors = ["แดง", "เขียว", "น้ำเงิน"]

# วนซ้ำแบบธรรมดา
for color in colors:
    print(color)

# วนซ้ำพร้อม index (เริ่มนับที่ 1)
for i, color in enumerate(colors, 1):
    print(f"{i}. {color}")

📋 ขั้นตอนที่ 4: List Comprehension

List Comprehension คือวิธีสร้าง List ใหม่จาก List เดิมในบรรทัดเดียวอย่างกระชับ มีรูปแบบ [นิพจน์ for ตัวแปร in iterable] และสามารถเพิ่มเงื่อนไขด้วย if ได้ เช่น [x*2 for x in nums if x > 0]

Python 
nums = [1, -2, 3, -4, 5, -6]

# คูณทุกค่าด้วย 2
doubled = [x * 2 for x in nums]
print(doubled)  # [2, -4, 6, -8, 10, -12]

# กรองเฉพาะค่าบวก
positives = [x for x in nums if x > 0]
print(positives)  # [1, 3, 5]

# สร้าง List กำลังสองของค่าบวก
pos_squared = [x**2 for x in nums if x > 0]
print(pos_squared)  # [1, 9, 25]
